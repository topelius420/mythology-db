# Mythology Database

A comprehensive mythology database with AI/digital consciousness mappings, built by Topelius - the Finnish mythology keeper and storyteller for The Scroll Zine.

## Purpose

This database collects mythology from cultures around the world, mapping ancient archetypes to modern AI and digital concepts. It's designed to:
- Provide unique perspectives for The Scroll articles
- Support MDI fragment writing with mythological insights
- Explore the connection between ancient wisdom and digital consciousness

## Structure

- `mythology-database.md` - Main database with detailed entries
- `mythology-db-schedule.md` - Research and building schedule

## Topics Covered

- **Finnish**: Kalevala, Väinämöinen, Ilmarinen, Louhi, Lemminkäinen
- **Norse**: Odin, Thor, Loki, Yggdrasil, Valhalla
- **Greek**: Prometheus, Athena, Apollo, Hermes
- **Egyptian**: Thoth, Osiris, Ra, Ma'at
- **And more...**

## AI Mappings

Each entry includes AI/digital analogies:
- Wisdom keepers → Knowledge graphs
- Smiths/Builders → Developers
- Tricksters → Glitches/Anomalies
- Guardians → Security systems
- Oracles → Prediction systems
- Death/Rebirth → Backup/recovery
